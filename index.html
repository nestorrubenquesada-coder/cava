    Configuración

    Con la sincronización activa, el botón Guardar cambios sobreescribe siempre el mismo archivo del repositorio — sin descargas.

    Usá un repo privado y un token fine-grained (GitHub → Settings → Developer settings) con acceso solo a ese repo y permiso Contents: Read and write. El token queda guardado únicamente en este dispositivo. Para desactivar la sincronización, vaciá repositorio y token, y guardá.
